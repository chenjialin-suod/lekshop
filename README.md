## LEKSHOP系统亮点
- 开源免费：LEKSHOP以开源方式发布, 允许用户自由使用、修改。
- 多商户系统：商户入驻, 后台审批。
- 逛一逛：短视频带货。
- 多语言支持：支持多种语言(简体,英,繁体)切换，我司有另外的<a href="https://www.lekshop.cn/16552.html">跨境电商系统, 前往体验</a>。
- 商城装修：支持商城自定义装修.
- 在线聊天：内置在线客服聊天系统

## 软件架构
Java语言开发，基于 SpringBoot 框架，前端 uniapp, 后台使用 vue-element-ui

## 环境要求
- 独立服务器(不能使用虚拟空间)
- CentOS 7.0+ 或 Ubuntu 20.04+
- Jdk8
- MySQL 8.0+
- Nginx 1.20+
- Redis
